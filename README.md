# Python Quiz Application

This is a console-based Quiz Application built using Python.

## Features
- Multiple difficulty levels
- Randomized questions
- Timer for each question
- Score tracking
- Leaderboard using file handling

## Technologies Used
- Python
- Random module
- Time module
- File handling

## How to Run

1. Download the file
2. Run the program

python quiz_application.py
